# myServiceBook

myServiceBook project for Cooperative IT Design course at KTH
  - Clone the project
  - Navigate to "frontend" folder
  - Run command "npm install"
  - Run command "npm run build"
  - Run command "npm run develop"
  - Use "localhost:3000" or "localhost:8080"
